# Email Template with DocPad

## Install

1. NPM and DocPad:

	```
	$ npm install -g npm;
	$ npm install -g docpad@6.78
	```

2. Clone project

4. Install dependencies and run:

    ```
    $ npm install
    $ docpad run
    ```
